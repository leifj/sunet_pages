cmu verifypkc

Verify a Public Key Confirmation from the HSM.   

Syntax

cmu verifypkc  <parameters>

Required Parameters

Parameter Description
-inputFile

 This parameter defines the name of the file that contains the PKC.
-pkctype

This parameter defines the PKC type (1 - TC-TrustCenter, 2 - Chrysalis-ITS).   

Optional Parameters

None.

Example

cmu verifypkc –inputFile=test.pkc –pkctype=1
