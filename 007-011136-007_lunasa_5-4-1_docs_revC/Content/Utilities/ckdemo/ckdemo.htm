CKdemo

This chapter describes how to access and use the ckdemo demonstration utility. The ckdemo utility is a simple console-based tool that provides a menu of functions that perform operations based on the PKCS#11 API. The ckdemo utility is included with the Luna client and can be used with any Luna HSM.
