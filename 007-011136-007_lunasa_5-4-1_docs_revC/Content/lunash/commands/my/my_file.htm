my file

Access commands that allow the currently logged in user to manage their files.

Syntax

my file

clear
delete
list
run

Parameter Shortcut Description
clear c Delete all of the files owned by the currently logged in user. See "my file clear".
delete d

Delete a file owned by the currently logged in user. See "my file delete".
list l

List the files owned by the currently logged in user. See "my file list".
run r Run a file owned by the currently logged in user as a Luna shell script. See "my file run".
