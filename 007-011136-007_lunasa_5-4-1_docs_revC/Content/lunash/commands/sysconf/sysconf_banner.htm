sysconf banner

Access the sysconf banner commands to set and clear an extended text banner, displayed to appliance administrative users when they log into a Luna Shell session.

Syntax

sysconf banner

add
clear

Parameter Shortcut      Description         
add a

Add extended banner text from a file. See "sysconf banner add" .
clear c Clear the extended banner text. See "sysconf banner clear".
