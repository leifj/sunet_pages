sysconf appliance

Access the sysconf appliance commands to manage the appliance.

Syntax

sysconf appliance

cpugovernor
hardreboot
poweroff
reboot
rebootonpanic
watchdog

Parameter Shortcut Description
cpugovernor c System CPU on-demand governor. See "sysconf appliance cpugovernor".
hardreboot h Reboot the appliance, bypassing graceful closing of services. See "sysconf appliance hardreboot".
poweroff p Power off the appliance. See "sysconf appliance poweroff".
reboot r Reboot the appliance. See "sysconf appliance reboot".
rebootonpanic rebooto System reboot on panic. See "sysconf appliance rebootonpanic ".
watchdog w System watchdog. See "sysconf appliance watchdog".
