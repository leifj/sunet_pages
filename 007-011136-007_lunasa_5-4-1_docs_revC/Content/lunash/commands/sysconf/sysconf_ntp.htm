sysconf ntp

Access the commands used to view and set the network time protocol (NTP) configuration.

Syntax

sysconf ntp

addserver
autokeyauth
deleteserver
disable
enable
listservers
log
ntpdate
show
status
symmetricauth

Parameter Shortcut Description
addserver ad Add NTP Server. See "sysconf ntp addserver".
autokeyauth au NTP Autokey Authentication. See "sysconf ntp autokeyauth".
deleteserver de Delete NTP Server. See "sysconf ntp deleteserver ".
disable di Disable NTP Service. See "sysconf ntp disable".
enable e Enable NTP Service. See "sysconf ntp enable".
listservers li List Configured NTP Servers. See "sysconf ntp listservers".
log lo

NTP Log Command. See "sysconf ntp log".
ntpdate n Set date and time using NTP. See "sysconf ntp ntpdate".
show sh Show NTP Configuration. See "sysconf ntp show".
status st Get NTP Service Status. See "sysconf ntp status"
symmetricauth sy NTP Symmetric Key Authentication. See "sysconf ntp symmetricauth".
