partition

Access commands that allow you to manage partitions on the appliance.

Syntax

partition

activate
backup
changepolicy
changepw
clear
create
createuser
deactivate
delete
list
resetpw
resize
restore
setlegacydomain
show
showcontent
showpolicies

Parameter Shortcut Description
activate a Activate a partition by caching its PED key data, allowing clients to authenticate with their partition password only. See "partition activate".
backup b Backup the partition to a backup token. See "partition backup".
changepolicy changepo Change the policies for a partition. See "partition changepolicy".
changepw changepw Change a partition password. See "partition changepw".
clear cl Delete all objects on a partition. See "partition clear".
create create Create an HSM partition on the HSM. See "partition create".
createuser createu Create a Crypto-User on a partition. See "partition createuser"
deactivate dea De-activate a partition by de-caching its PED key data, so that clients can no longer authenticate with their partition password only. See "partition deactivate".
delete del Delete an HSM partition from the HSM. See "partition delete".
list l Display a list of the accessible partitions. See "partition list".
resetpw rese Reset a Partition Owner's password. See "partition resetpw".
resize resi Resizes the storage space for a partition. See "partition resize".
restore rest Restore the HSM partition contents from PCMCIA backup token. See "partition restore".
setlegacydomain se Set the legacy cloning domain on a partition. See "partition setlegacydomain".
show sh Display information for a partition. See "partition show".
showcontents showc Display a list of the objects on a partition. See "partition showcontents".
showpolicies showp Displays the policy configuration for a partition. See "partition showpolicies".
