network

Access commands that allow you to view and configure the network settings for the appliance.

Syntax

network

dns
domain
hostname
interface
ping
route
show

Parameter Shortcut Description
dns dn Access commands that allow you to configure the appliance DNS settings. See "network dns".
domain    do Set the network domain. See "network domain".
hostname   h Set the appliance hostname. See "network hostname".
interface   i Configure the network interfaces. See "network interface".
ping p Test the network connectivity. See "network ping".
route   r Access commands that allow you to configure the network routes for the appliance. See "network route".
show   s Display the current network configuration. See "network show".
