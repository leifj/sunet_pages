package

Access commands that allow you to manage secure package updates. Use these commands after you have copied the package files to the Luna SA, using the scp utility.

Syntax

package

deletefile
erase
list
listfile
update
verify

Parameter Shortcut Description
deletefile   d Delete a package file. See "package deletefile".
erase     e Delete a package . See "package erase".
list    l List the installed packages. See "package list".
listfile   listf List the uninstalled package files. See "package listfile".
update   u Update the package file. See "package update".
verify   v Verify the package file. See "package verify".
