htl set

Access commands that allow you to configure the attributes for an HTL client one-time token.

Syntax

htl set

defaultottexpiry
graceperiod
ottexpiry

Parameter Shortcut Description
defaultottexpiry d Sets the HTL one-time token default expiry time for a client. See "htl set defaultottexpiry".
graceperiod g

Sets the HTL grace period. See "htl set graceperiod".
ottexpiry o

Set the HTL one-time token expiry time for a client. See "htl set ottexpiry".
