ntls information

Access commands hat allow you to display information about the NTLS connection or reset the NTLS counters.

Syntax

ntls information

reset
show

Parameter Shortcut Description 
reset  r Reset the NTLS counters. See "ntls information reset".
show  s Display NTLS information. See "ntls information show".
