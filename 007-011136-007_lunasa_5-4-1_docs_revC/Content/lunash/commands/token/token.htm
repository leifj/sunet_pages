LunaSH Command Reference Guide > LunaSH Commands > token

token

Access the token-level commands. These commands are separate menus for token HSMs as backup devices or token HSMs used in PKI mode.

Syntax

token

backup
pki

Parameter Shortcut Description
backup b Access the token backup commands. See "token backup".
pki p Access the token pki commands. See "token pki".
