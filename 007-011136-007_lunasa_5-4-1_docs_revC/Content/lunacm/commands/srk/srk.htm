LunaCM Command Reference Guide > LunaCM Commands > srk

srk

Access the srk-level commands. These commands manage the secure recovery key (SRK) behavior and the setting and recovery of Secure Transport Mode. See MTK and SRK discussion here.

Syntax

srk

disable
enable
generate
recover
show
transport

Parameter Shortcut Description
disable
d Disable Secure Transport Mode functionality. See "srk disable".
enable e Enable Secure Transport Mode functionality. See "srk enable".
generate g Generate a new SRK. See "srk generate".
recover r Recover from temper or exit transport mode. See "srk recover".
show s Show the Secure Recovery state. See "srk show".
transport t Set the HSM into transport mode. See "srk transport".
