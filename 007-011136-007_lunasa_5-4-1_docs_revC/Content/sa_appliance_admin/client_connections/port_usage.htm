Luna SA Port Usage

Here is how ports are used on the Luna SA appliance, by default.

Standard Ports

Port Type Port Usage Direction
TCP 22 SSH (Secure Shell)   
Network Access to appliance from client and/or remote workstations  for administration 		Bi-directional   
TCP 1792 NTLS (Network Trust Link Service)   
Application traffic   
Luna Client Utilities cmu, vtl, your application(s), etc.  [*] 		 Bi-directional   
TCP 1503 RemotePED   
Only port that is configurable   
Establishing secure connection for a Remote PED   
Not applicable in a PWD based HSM   		 HSM to Remote Workstation/Client   

[* Luna SA HSM communicates with the Luna Client. Applications use the client connection to obtain service from the HSM. Service is available only to client systems that are registered with Luna SA HSM partitions.]

Additional Ports

Port Type Port Usage Direction
UDP 514 Syslog Service
Used to offload syslog to a remote syslog server 		HSM to Syslog Server
UDP 123 NTP Service (Network Time Protocol) HSM to NTP Server
UDP 161/162 SNMP Service
(Simple Network Management Protocol) 		HSM to SNMP Server
