Luna SA 5.4.1

PRODUCT DOCUMENTATION

The Luna SA documentation is provided in HTML and PDF formats. You can access it directly from the DVD, or you can copy it to a file server or web server for distributed access. See the README for more information. To fully understand the capabilities, limitations, and known issues for this release, refer to the Customer Release Notes.

HTML PDF Description
Product Overview Provides an overview of the Luna HSM product line.
Installation Guide Describes how to install the Luna SA, PCI-E, and G5 HSMs, the Luna Remote PED, and the Luna Client software.
Configuration Guide Describes how to configure your Luna HSM to prepare it to work in your environment.
HSM Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your HSMs.
Appliance Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your Luna SA appliance.
LunaCM Command Reference Guide Describes how to access and use the LunaCM command line tool and provides detailed syntax descriptions for each available command.
LunaSH Command Reference Guide Describes how to access and use the LunaSH command line tool and provides detailed syntax descriptions for each available command.
Utilities Reference Guide Describes the various utilities included with the Luna client software.
SDK Reference Guide Describes how to use the Luna SDK to integrate your applications with a Luna HSM.
Syslog Monitoring Guide (PDF only) Describes the syslog facility and provides details for what various messages mean, the probability of observing them, and what action to take if a message is observed.
