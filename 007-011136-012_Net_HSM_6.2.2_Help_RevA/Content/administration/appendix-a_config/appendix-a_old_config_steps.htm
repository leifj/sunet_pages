Appendix A: Configuration Long-form

This section documents the long-form, detailed, step-by-step instructions for configuring your SafeNet HSM hardware, before you begin using it with your application(s). These instructions do not make use of the One-step NTLS Setup that was introduced in release 6.2.1.

To ensure a trouble-free configuration, perform the following steps in the order indicated:

1.[Step 1] Planning Your Configuration

2.[Step 2] Configure Your Network Settings

3.[Step 3] Initialize the HSM

4.[Step 4] Set the HSM Policies

5.[Step 5] Create Application Partitions

6.[Step 6] Set the Partition Policies for Legacy Partitions

7.[Step 7] Create a Network Trust Link Between the Client and the Appliance

8.[Step 8] Enable the Client to Access a Partition

9.[Step 9] Configure PPSO Application Partitions

10.[Step 10] Set the Partition Policies for PPSO Partitions

Also review Optional Configuration Tasks for more configuration options.
