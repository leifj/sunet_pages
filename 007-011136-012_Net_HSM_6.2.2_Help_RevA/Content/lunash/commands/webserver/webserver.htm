webserver

The webserver command set is available in LunaSH (lunash:>) if your SafeNet Network HSM appliance is at version 6.0 or higher, and you have the REST API configuration upgrade installed.

Syntax

webserver

enable
disable
certificate
bind
ciphers
show

Parameter   Shortcut   Description  
enable e   Enable REST API service. See webserver enable. 
disable  d   Disable REST API service. See webserver disable .  
certificate   ce   Manage REST API service certificate. See webserver certificate  .   
bind   b  Set REST API service network port. See webserver bind  .   
ciphers ci   Manage REST API service cipher suite.  See webserver ciphers  .  
show   s   Show REST API service configuration and status. See webserver show  .
