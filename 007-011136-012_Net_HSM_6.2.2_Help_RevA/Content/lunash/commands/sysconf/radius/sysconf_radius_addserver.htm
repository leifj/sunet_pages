sysconf radius addserver Command

Identify a RADIUS server to the SafeNet Network HSM, specifying the server's hostname or IP.

Note:  RADIUS must already be enabled, by means of command sysconf radius enable, before you can run this command to add a RADIUS server.

In addition to enabling RADIUS, you must run the sysconf radius addServer command to identify the RADIUS server, as well as the user role radiusAdd and user role add commands to create a user on this appliance with the desired name, and identify that role as requiring RADIUS to authenticate.

Syntax

sysconf radius addServer  -server <hosthame> -port <port> -timeout <seconds>

Parameter                Shortcut Description   
-server    -s<hostname> Host name  
-port    -p<port> Network port (0-65535)  
-timeout    -t<seconds> Time in seconds (1 - 300)

Example

lunash:>sysconf radius addServer -server 192.20.15.182 -port 1812 -timeout 60
