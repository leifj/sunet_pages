sysconf

Access commands that allow you to configure the appliance.

Syntax

sysconf

appliance
config
drift
fingerprint
hwregencert
ntp
radius
regencert
securekeys
snmp
ssh
time
timezone

Parameter Shortcut    Description   
appliance   a    Access commands that allow you to manage the appliance. See sysconf appliance.
config c    Access the system configuration commands. See sysconf config.
drift d    Access commands that allow you to view and configure the drift. See sysconf drift.
fingerprint   f    Display the certificate fingerprints. See sysconf fingerprint.
hwregencert     h    Generate or re-generate the SafeNet appliance server hardware certificate . See sysconf hwregencert
ntp   n    Access commands that allow you to view or configure the network time protocol (NTP). See sysconf ntp.
radius     ra    Manage RADIUS configuration and identify RADIUS servers to use for enhanced authentication, authorization, and accounting of your SafeNet appliance users and roles sysconf radius .
regenCert    re    Generate or re-generate the SafeNet appliance server hardware certificate. See sysconf regencert.
securekeys se    Move the SafeNet keys used to secure the NTLS link from the SafeNet appliance's file system into the HSM. See sysconf securekeys.
snmp sn    Access commands that allow you to view or configure the Simple Network Management Protocol (SNMP) settings for SafeNet appliance. See sysconf snmp.
ssh ss   

Access commands that allow you to view or configure the SSH options on the appliance. See sysconf ssh.
time    t    Set or display the time and date. See sysconf time.
timezone timez    Set or display the time zone. See sysconf timezone.
