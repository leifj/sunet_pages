sysconf appliance rebootonpanic

Access commands that allow you to enable or disable reboot on panic and show reboot on panic information.

Syntax

sysconf appliance rebootonpanic

disable
enable
show

The following subcommands are available:  

Parameter Shortcut Description   
disable   d    Disable system reboot on panic. See sysconf appliance rebootonpanic disable.
enable    e   Enable system reboot on panic. See sysconf appliance rebootonpanic enable.
show s    Show system reboot on panic information. See sysconf appliance rebootonpanic show .
