hsm srk transportmode

Access commands that allow you to put the HSM into, or out of, secure transport mode.

Syntax

hsm srk transportmode

enter
recover

Parameter Shortcut Description
enter e Places the HSM in Transport Mode. See hsm srk transportmode enter.
recover r Takes the HSM out of Transport Mode. See hsm srk transportmode recover.
