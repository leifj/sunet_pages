hsm srk keys

Access commands that allow you to resplit or verify secure recovery keys (SRK).

Syntax

hsm srk keys

resplit
verify

Parameter Shortcut Description
resplit r Re-splits the Secure Recovery Key. See  hsm srk keys resplit
verify v Verifies an existing Secure Recovery Key. See hsm srk keys verify.
