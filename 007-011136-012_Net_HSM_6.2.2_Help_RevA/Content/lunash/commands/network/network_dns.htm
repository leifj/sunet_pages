network dns

Access commands that allow you to configure the appliance DNS settings.

Syntax

network dns

add
delete

 

Parameter Shortcut Description
add a Add domain name servers and search domains to the network configuration. See network dns add.
delete    d Delete domain name servers and search domains from the network configuration. See network dns delete.
