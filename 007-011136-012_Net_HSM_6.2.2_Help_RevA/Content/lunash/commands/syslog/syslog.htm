syslog

Access the syslog commands used to manage the system logs.

Note:  Syslog format is in accordance with RFC 5424.

Note:  Syslog uses system time. If you change the timezone setting for the appliance while syslog is running, syslog continues to log entries based on the old timezone, until you restart the syslog service.

Syntax

syslog

cleanup
export
period
policy
remotehost
rotate
rotations
severity
show
tail
tarlogs

Parameter Shortcut Description
cleanup c Delete log files. See sylog cleanup.
export e Export syslog. See syslog export.
period p Set the syslog period. See syslog period.
remotehost re Configure Syslog remote hosts. See syslog remotehost.
rotate rotate Rotate log files. See syslog rotate.
rotations rotati Set syslog rotations. See syslog rotations
severity se Log severity. See syslog severity set.
show sh Get Syslog configuration. See syslog show.
tail tai Get last entries of log. See syslog tail.
tarlogs tar Archive log files. See syslog tarlogs.
