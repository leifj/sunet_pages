SafeNet USB HSM Hardware Installation

This chapter describes how to install and connect a SafeNet USB HSM. To ensure a successful installation, perform the following tasks in the order indicated:

1.Ensure that you have all of the required components, as listed in SafeNet USB HSM Required Items

2.Install the included removable battery, as described in Installing the Battery in the SafeNet USB HSM or Backup HSM. The SafeNet USB HSM requires the battery to operate.

3.Install and connect the hardware, as described in Installing the SafeNet USB HSM Hardware
