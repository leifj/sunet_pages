appid

Access the appid-level commands to manage application IDs on the HSM.

Syntax

appid

open
close
set
info

Parameter Shortcut Description
open o Open a previously set access ID. See appid open
close c Close a previously set access ID. See appid close
set s Set an access ID. See appid set
info i Display information for the access IDs. See appid info

Example

lunacm:> help appid
The following sub commands are available:
Command         Short    Description
------------------------------------
open            o        Open an Application Id for the User
close           c        Close an Application Id for the User
set             s        Set the Application Id
info            i        Display current Application Id information
Syntax: appid <sub command>
Command Result : No Error
