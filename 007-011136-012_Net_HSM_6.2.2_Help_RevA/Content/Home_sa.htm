Product Documentation

The SafeNet Network HSM documentation is provided in HTML and PDF formats. You can access it directly from the DVD or downloaded archive, or you can copy it to a file server or web server for distributed access. See the README for more information.

HTML PDF Description
Product Overview Provides an overview of the SafeNet HSM product line.
Installation Guide Describes how to install the SafeNet Network, PCIe, and USB HSMs, the Remote PED, and the SafeNet HSM client software.
Configuration Guide Describes how to configure your SafeNet HSM to prepare it to work in your environment.
HSM Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your HSMs.
Appliance Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your SafeNet Network HSM appliance.
LunaCM Command Reference Guide Describes how to access and use the LunaCM command line tool and provides detailed syntax descriptions for each available command.
LunaSH Command Reference Guide Describes how to access and use the LunaSH command line tool and provides detailed syntax descriptions for each available command.
Utilities Reference Guide Describes the various utilities included with the SafeNet HSM client software.
SDK Reference Guide Describes how to use the SafeNet HSM SDK to integrate your applications with a SafeNet HSM.
Syslog and SNMP Monitoring Guide (PDF only) Describes the syslog facility and provides details for what various messages mean, the probability of observing them, what action to take if a message is observed, and how to map the syslog messages to SNMP traps.
