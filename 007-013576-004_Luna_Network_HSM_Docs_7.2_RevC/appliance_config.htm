Configure the SafeNet Luna Network HSM for Your Network

This chapter describes how to configure your SafeNet Luna Network HSM appliance so that you can access it over the network. This involves performing the following tasks, in the order specified:

2.Open a Connection

3.Logging In to the Appliance

4.Network Configuration

5.Make Your Network Connection

6.Set TLS ciphers

7.Set the System Date and Time

8.Generating the HSM Server Certificate

9.Binding Your NTLS or SSH Traffic to a Device
