sysconf ntp autokeyauth update

Update the client certificates and keys.

User Privileges

Users with the following privileges can perform this command:

>Admin

>Operator

Syntax

sysconf ntp autokeyauth update

Example

lunash:>sysconf ntp autokeyAuth update
