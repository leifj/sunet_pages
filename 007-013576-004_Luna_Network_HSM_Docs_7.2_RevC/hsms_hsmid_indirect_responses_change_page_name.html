POST /api/lunasa/hsms/{hsmid}/indirect/responses

Gets the indirect login response needed for the service HSM from the administration partition on the admin HSM.

Parameters

hsmid

The serial number of the HSM of interest

Use: Required

JSON Schema: 

   Object
   type: string

challenge

The challenge needed to authenticate to the service HSM

Use: Required

JSON Schema: 

   Object
   type: string

Responses

201

The response to provide to the service HSM for indirect login.

JSON Schema: 

    type:  string
    description:  response is the authentication data to be provided for indirect login
                  to the service HSM.

Location

"Location" is the URL to the response and is returned in the server response. You can use "Location" to form a GET resource to obtain the response.

see GET /api/lunasa/hsms/{hsmid}/indirect/responses/{responseid}

400

Unexpected error
