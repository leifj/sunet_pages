sysconf config

Access the system configuration commands. This command manages the various configuration files that are created and modified when you set up various system elements such as NTLS, SSH, NTP, SNMP, etc.

Syntax

sysconf config

backup
clear
delete
export
factoryreset
import
list
restore
show

Argument(s) Shortcut Description
backup b Backs up configuration data. See sysconf config backup.
clear c Deletes all the configuration backup files except the initial factory configuration file. See sysconf config clear.
delete d Deletes a configuration backup file. See sysconf config delete.
export e Exports a configuration backup file. See sysconf config export.
factoryreset f Factory reset. See sysconf config factoryreset.
import i Imports a configuration backup file. See sysconf config import.
list l List configuration backup files. See sysconf config list.
restore r Restores configuration backup. See sysconf config restore.
show s Show the current configuration. See sysconf config show.
