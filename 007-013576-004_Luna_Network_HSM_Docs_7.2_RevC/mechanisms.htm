Supported Mechanisms

This chapter provides an alphabetical listing of the supported PKCS#11 standard mechanisms and SafeNet proprietary mechanisms supported in firmware 7.2.0.

See Mechanism Summary for a table summarizing all of the supported mechanisms, or refer to the individual page for the mechanism you are interested in.

If you are using RSA keys in FIPS mode, refer to Mechanism Remap for FIPS Compliance for information about remapping your mechanisms to maintain FIPS compliance.
