vtl verify

Name

vtl verify

Syntax

vtl verify

Description

Verify the SafeNet Luna Network HSM slots/Partitions visible to this Client.

Options

None.

Example

            bash-2.03# ./vtl verify
        
The following SafeNet Luna Network HSM Slots/Partitions were found:
Slot    Serial #        Label
====    ========        =====
 1      65091001        MyPartition
 2      65097001        YourPartition
 3      65093001        HisPartition  
bash-2.03#
