appid info

Display the currently set application IDs. This list includes all set application IDs, regardless of whether they are open or closed. For a full description of application IDs, see Application IDs in the SDK Reference Guide.

Syntax

appid info

Example

lunacm:>appid info
        Using user defined Application ID:
 
        Application ID Major: 307
        Application ID Minor: 207
 
Command Result : No Error
