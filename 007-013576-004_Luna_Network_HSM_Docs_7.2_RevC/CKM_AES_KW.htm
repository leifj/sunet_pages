CKM_AES_KW

NIST Special Publication 800-38F describes cryptographic methods that are approved for “key wrapping,” that is, the protection of the confidentiality and integrity of cryptographic keys. In addition to describing existing methods, that publication specifies two new, deterministic authenticated-encryption modes of operation of the Advanced Encryption Standard (AES) algorithm: the AES Key Wrap (KW) mode and the AES Key Wrap With Padding (KWP) mode. Gemalto's SafeNet Luna Network HSMs implement the AES Key Wrap (KW) mode at this time, which SP800-38F recommends as more secure than CKM_AES_CBC.

NOTE   NIST Special Publication 800-38F recommends this method as more secure than CKM_AES_CBC.
