hsm firmware

Upgrade to the version of HSM firmware that is currently on standby in the SafeNet Luna Network HSM appliance.

Rollback to the previous version of HSM firmware, retained in the SafeNet Luna Network HSM appliance.

Syntax

hsm firmware

rollback
upgrade
show

Argument(s)

Shortcut

Description

show s

Show HSM firmware version info. See hsm firmware show.
upgrade u

Update HSM firmware. See hsm firmware upgrade.
rollback r Rollback HSM firmware. See hsm firmware rollback.
