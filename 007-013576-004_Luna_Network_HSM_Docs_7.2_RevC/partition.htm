partition

Access commands used to manage partitions on the HSM. These commands are used by the HSM SO to create. delete, or resize partitions on the HSM. The partitions are owned by the Partition SO, and configured using LunaCM.

Syntax

partition

backup
create
delete
list
rename
resize
restore
show

Argument(s) Shortcut Description

backup

b

Backup the contents of an HSM partition to a backup HSM. See partition backup.

create

c

Create an HSM partition on the HSM. See partition create.

delete

d

Delete an HSM partition from the HSM. See partition delete.

list

l

Display a list of the accessible partitions. See partition list.

rename

ren

Renames the specified partition. See partition rename.

resize

resi

Resizes the storage space for a partition. See partition resize.

restore

rest

Restore the contents of an HSM partition from a backup HSM. See partition restore.

show

s

Display information for a partition. See partition show.
