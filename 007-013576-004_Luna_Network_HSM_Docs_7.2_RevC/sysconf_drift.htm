sysconf drift

Access the sysconf drift commands to view and configure drift.

Syntax

sysconf drift

init
reset
set
startmeasure
status
stopmeasure

Argument(s) Shortcut Description
init i

Activate automatic drift adjustments. See sysconf drift init.
reset r Reset all drift tracking data. See sysconf drift reset.
set se

Manually set internal drift data. See sysconf drift set.
startmeasure star Set the time and start measuring. See sysconf drift startmeasure.
status stat Display the current drift data. See sysconf drift status.
stopmeasure sto Stop measuring and record the drift. See sysconf drift stopmeasure.
