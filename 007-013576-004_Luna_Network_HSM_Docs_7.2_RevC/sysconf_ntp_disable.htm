sysconf ntp disable

Disable and stop the NTP service.

User Privileges

Users with the following privileges can perform this command:

>Admin

>Operator

Syntax

sysconf ntp disable

Example

lunash:>sysconf ntp disable
 
NTP is disabled
Stopping ntpd:                                             [  OK  ]
NTP is stopped
 
Command Result : 0 (Success)
