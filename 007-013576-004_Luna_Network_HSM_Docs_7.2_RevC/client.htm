client

Access commands that allow you to manage the SafeNet Luna HSM clients that are able to use partitions on the appliance.

Syntax

client

assignpartition
delete
fingerprint
hostip
list
register
revokepartition
show

Argument(s) Shortcut Description
assignpartition a Assign partition access rights to a client. See client assignpartition.
delete d Delete a client. See client delete.
fingerprint f Display the certificate fingerprint for a registered client. See client fingerprint.
hostip h Display or configure the client-to-IP mapping. See client hostip.
list l Display a list of the registered clients by client name. See client list.
register reg Add a client to the list of clients that can access the SafeNet appliance's NTLS. See client register.
revokepartition rev Revoke access privileges to the specified partition from the specified client. See client revokepartition.
show s Display the hostname or IP address of a client, and any partitions assigned to the client. See client show.
