Release Notes

Product Documentation

The SafeNet Luna Network HSM 7.2 documentation, linked below, is provided in HTML and PDF formats. See the README for more information. You can download the latest version from the Gemalto eService Support Portal at https://supportportal.gemalto.com.

Overview, Installation, and Migration
Product Overview Describes what an HSM is, and its major use cases. Provides an overview of the SafeNet Luna HSM product line, and describes its key features and benefits, focusing on key differentiators.
Installation Guide Describes how to install the SafeNet Luna Network, PCIe, and USB HSMs, the Remote PED, and the SafeNet Luna HSM Client software.
Key Migration Guide Describes how to migrate your cryptographic objects from a legacy SafeNet Luna HSM to a Release 7 SafeNet Luna HSM.
Configuration and Administration
Configuration Guide Describes how to configure your SafeNet Luna Network HSM to prepare it to work in your environment.
HSM Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your HSMs.
Appliance Administration Guide Describes how to perform ongoing operational and administrative tasks to maintain the operational functionality and efficiency of your SafeNet Luna Network HSM appliance.
Tools and Utilities
LunaCM Command Reference Guide Describes how to access and use the LunaCM command line tool and provides detailed syntax descriptions for each available command.
LunaSH Command Reference Guide Describes how to access and use the LunaSH command line tool and provides detailed syntax descriptions for each available command.
REST API User Guide Describes the client and provides some instructions and examples for installing and setting up the API.
REST API Reference Guide Provides full descriptions of operations and calls available through the API.
Utilities Reference Guide Describes the various utilities included with the SafeNet Luna HSM Client software.
Performance Monitoring
Syslog and SNMP Monitoring Guide Describes the syslog facility and provides details for what various messages mean, the probability of observing them, what action to take if a message is observed, and how to map the syslog messages to SNMP traps.
Application Integration
SDK Reference Guide Describes how to use the SafeNet Luna HSM SDK to integrate your applications with a SafeNet Luna HSM.
