Detailed Description

This class calculates a hash value.

The following hash algorithms are supported (see Hash Algorithms):

  • SHA-1
  • RipeMD160
  • MD5
  • SHA224
  • SHA256
  • SHA384
  • SHA512
Note
The hash value is calculated locally.
Alternatively a hash value may also calculated on the CryptoServer (see CryptoServerCXI::Cxi::computeHash).
Example (C#):

The following code sample illustrates how to calculate a hash value:

Hash hash = new Hash(Hash.ALGO_SHA256);
hash.update(data);
hash.update(...);
byte [] hashValue = hash.final();
