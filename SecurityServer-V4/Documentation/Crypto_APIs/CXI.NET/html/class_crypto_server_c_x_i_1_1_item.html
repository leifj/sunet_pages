Detailed Description

This class provides methods to handle with TLV coded items.

An item is constructed as follows:

+---------+---------+-------------+
| TAG | LEN | VALUE |
+---------+---------+-------------+
| 2 Bytes | 4 Bytes | <LEN> Bytes |
+---------+---------+-------------+
Note
Items are used to build command and answer data blocks or key blobs.
Example (C#)

The following example illustrates how to extract key properties from a key blob (see Format of Key Blobs):

byte [] list = Item(keyBlob).find("KB").find("PL").getValue();
KeyProperties propList = new KeyProperties(list);
