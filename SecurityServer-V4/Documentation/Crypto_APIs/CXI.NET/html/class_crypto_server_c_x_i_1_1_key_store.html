Detailed Description

Key storage for external (MBK encrypted) keys.

On creation or import of a key on the CryptoServer, it can either be stored on the CryptoServer or exported as an external key (see Cxi::FLAG_KEY_EXTERNAL).
In the latter case the key is encrypted with the CryptoServer's Master Box Key (MBK). Such external keys can be stored in an external key database provided by this class.

Each record in a key store database consists of two fields:

  • A search key (index) that uniquely identifies the record within the database.
  • A data field where the key blob is stored.

The index fields of all records have the same length. This length has to be defined on creation of the database and can't be changed later.
Most commonly the unique key name (MD5 hash over name, group and specifier) with a length of 16 bytes is used as index (see KeyProperties::getUName()).
But the application may use any other index (e.g. a continuous serial number) to identify a key.

The length of the data field is variable.

Example:

The following example illustrates the usage of the KeyStore class:

KeyStore ks = new KeyStore("cxi.ks", 16);
// create RSA key (external key storage)
KeyProperties keyTemplate = new KeyProperties();
keyTemplate.setGroup("MyKeyGroup");
keyTemplate.setName("MyRSAKey");
keyTemplate.setAlgo(Cxi.KEY_ALGO_RSA);
keyTemplate.setSize(1024);
byte [] keyBlob = cxi.generateKey(Cxi.KEY_FLAG_EXTERNAL, keyTemplate, -1);
// get original database index (unique name property) from CryptoServer
KeyProperties propList = cxi.getKeyProperties(keyBlob, int [] { Cxi.PROP_KEY_UNAME } );
// store key in external key store
ks.insertKey(KeyStore.FLAG_OVERWRITE, propList.getUName(), keyBlob);
// get AES key
keyTemplate = new KeyProperties();
keyTemplate.setGroup("MyKeyGroup");
keyTemplate.setName("MyAESKey");
keyBlob = cxi.findKey(Cxi.KEY_FLAG_EXTERNAL, keyTemplate);
// get datebase index (unique name property) from key blob
propList = new KeyProperties(Item(keyBlob).find("KB").find("PL").getValue());
// store key in external key store
ks.insertKey(KeyStore.FLAG_OVERWRITE, propList.getUName(), keyBlob);
// list only RSA keys on first loop
keyTemplate = new KeyProperties();
keyTemplate.setAlgo(Cxi.KEY_ALGO_RSA);
for (int i=0; i<2; i++)
{
byte [] index = new byte [ks.getIndexLength()];
KeyStore.FindMode mode = KeyStore.FindMode.GtEq;
Console.WriteLine(String::Format("\n{0,-5} {1,-5} {2,-24} {3,-24}", "algo", "size", "group", "name"));
Console.WriteLine("--------------------------------------------------------------------------------");
while (ks.findKey(index, mode, keyTemplate) == true)
{
keyBlob = ks.getKey(index);
// get property list form key blob
propList = new KeyProperties(new Item(keyBlob).find("KB").find("PL").getValue());
String s = String.Format("{0,-5} {1,-5:d} {2,-24} {3,-24}",
propList.getAlgoName(),
propList.getSize(),
propList.getGroup(),
propList.getName() );
Console.WriteLine(s);
mode = KeyStore.FindMode.Greater;
}
// list all keys on second loop
keyTemplate = null;
}
