Detailed Description

This class provides methods to create and handle key property lists.

Key properties contain additional information stored together with the key. Some properties are mandatory (e.g. algorithm, size), other properties are optional (e.g. label, certificate). Some properties are applicable to all keys (e.g. name, usage), other properties are only used with special keys (e.g. modulus -> RSA key, curve -> ECDA key).
